# 接口契约

> 本文件是前后端接口联调的**唯一事实来源**。所有接口的路径、字段命名、类型、错误码以本文件为准。
> 前后端 spec 中涉及接口调用的部分，应引用本文件而非各自独立描述。
>
> 如果项目有前后端联调需求，将此文件重命名为 `api-contracts.md` 并填充实际接口。

## 全局约定

### 基础信息

- **Base URL**: `{环境变量名或占位符}`
- **协议**: HTTPS
- **内容类型**: `application/json`（除文件上传外）
- **字符编码**: UTF-8
- **字段命名风格**: camelCase（前后端统一，以本文件定义为准）

### 鉴权方式

- **方式**: <!-- Bearer Token / Cookie / API Key -->
- **Header**: `Authorization: Bearer {token}`
- **过期处理**: <!-- 401 时前端跳转登录页 / 自动刷新 token -->

### 通用响应结构

| 字段 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| `code` | `integer` | Y | 业务状态码，0 表示成功 |
| `message` | `string` | Y | 状态描述 |
| `data` | `object \| array \| null` | Y | 业务数据 |

### 通用错误码

| 错误码 | HTTP 状态码 | 含义 | 前端处理 |
|--------|:-----------:|------|----------|
| `0` | 200 | 成功 | 正常处理 |
| `401` | 401 | 未认证 | 跳转登录 |
| `403` | 403 | 无权限 | 提示无权限 |
| `404` | 404 | 资源不存在 | 提示不存在 |
| `422` | 422 | 参数校验失败 | 显示字段级错误 |
| `500` | 500 | 服务器内部错误 | 提示稍后重试 |

### 分页约定

请求参数（query string）：

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `page` | `integer` | `1` | 页码，从 1 开始 |
| `pageSize` | `integer` | `20` | 每页条数 |

响应分页信息（`data` 内包裹）：

| 字段 | 类型 | 说明 |
|------|------|------|
| `list` | `array` | 当前页数据 |
| `total` | `integer` | 总记录数 |
| `page` | `integer` | 当前页码 |
| `pageSize` | `integer` | 每页条数 |
